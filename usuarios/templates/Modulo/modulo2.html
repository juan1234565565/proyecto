¿Qué es un Sistema Operativo?

El sistema operativo (SO) es el software más importante de una computadora. Actúa como intermediario entre el usuario y el hardware, gestionando los recursos del sistema y permitiendo la ejecución de otros programas.

Funciones principales de un SO:

  • Gestión de procesos: Controla qué programas se ejecutan y cuándo
  • Gestión de memoria: Administra la memoria RAM
  • Gestión de archivos: Organiza y controla el acceso a archivos
  • Control de dispositivos: Gestiona periféricos (teclado, mouse, impresora)
  • Interfaz de usuario: Permite la interacción con el usuario

Sistemas Operativos Principales

  • El más utilizado en computadoras de escritorio
  • Interfaz gráfica amigable

Manejo Básico de Archivos

¿Qué es un archivo?

Un archivo es una colección de datos almacenados con un nombre específico. Puede contener texto, imágenes, programas, etc.

Tipos comunes de archivos:

📄
.txt, .doc, .pdf

Documentos de texto

🖼️
.jpg, .png, .gif

Imágenes

🎵
.mp3, .wav

Archivos de audio

🎬
.mp4, .avi

Archivos de video

⚙️
.exe, .msi

Archivos ejecutables

📦
.zip, .rar

Archivos comprimidos

Organización de Archivos y Carpetas

Estructura jerárquica:

  • Unidades: C:\, D:\ (en Windows)
  • Carpetas/Directorios: Contenedores para organizar archivos
  • Subcarpetas: Carpetas dentro de otras carpetas
  • Archivos: Los elementos finales que contienen datos

💡 Buenas prácticas de organización

  • Crea carpetas por proyecto o categoría
  • Usa nombres descriptivos para archivos y carpetas
  • Mantén tu escritorio organizado
  • Realiza copias de seguridad regularmente

Operaciones Básicas con Archivos

Crear:

  • Clic derecho → Nuevo → Carpeta/Archivo
  • O desde el menú "Archivo" en programas

Copiar y Mover:

  • Copiar: Ctrl+C → Navegar → Ctrl+V
  • Mover: Ctrl+X → Navegar → Ctrl+V
  • Arrastrar: Mantener Ctrl para copiar, Shift para mover

Eliminar:

  • Papelera de reciclaje: Archivos eliminados temporalmente
  • Shift+Delete: Eliminación permanente
  • Vaciar papelera: Eliminación definitiva

Buscar:

  • Windows: Presiona la tecla Windows y escribe
  • Explorador de archivos: Usa la barra de búsqueda
  • Mac: Spotlight (Cmd+Espacio)

Atajos de Teclado Útiles

Navegación general:

  • Ctrl+C / Cmd+C: Copiar
  • Ctrl+V / Cmd+V: Pegar
  • Ctrl+X / Cmd+X: Cortar
  • Ctrl+Z / Cmd+Z: Deshacer
  • Ctrl+A / Cmd+A: Seleccionar todo

Gestor de archivos:

  • Windows+E: Abrir Explorador de archivos
  • F2: Renombrar archivo/carpeta
  • F5: Actualizar vista
  • Ctrl+Shift+N: Nueva carpeta
  • Alt+F4: Cerrar ventana

Mantenimiento del Sistema

Tareas importantes:

  • Actualizaciones: Mantener el SO y programas actualizados
  • Limpieza de disco: Eliminar archivos temporales
  • Desfragmentación: Optimizar el disco duro (HDD)
  • Copias de seguridad: Resguardar datos importantes
  • Antivirus: Protección contra malware

⚠️ Precauciones de seguridad

  • No descargues archivos de fuentes no confiables
  • Cierra sesión en computadoras compartidas
  • Usa contraseñas seguras
  • Realiza escaneos periódicos con antivirus
